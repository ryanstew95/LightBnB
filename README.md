# LightBnB

## Project summary

A multi-page Airbnb clone that uses a server-side Javascript to display the information from queries to web pages via SQL queries.

![front page](img/lightbnb.png)

## Setup

Install dependencies with `npm install`.

## Dependencies

    -bcrypt,
    -cookie-session,
    -express,
    -nodemon,
    -pg

## How to run app

-In LighBnB_WebApp
`npm run local`

-In web browser
`localhost:3000`
